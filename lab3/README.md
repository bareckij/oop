# RPG Inventory System (C#)

Лабораторная работа: система управления инвентарём с применением SOLID и паттернов проектирования (Builder, Strategy, State, Abstract Factory).

## Структура

- `InventorySystem/` — основной проект (классы доменной логики)
- `InventorySystem.Tests/` — модульные тесты (xUnit)

## Требования

- .NET 8 SDK (можно использовать 7, если требуется, но пример ориентирован на 8)

## Как запустить тесты

```bash
cd lab3
# создание решения и проектов будет добавлено кодом помощника
# после генерации проектов:
dotnet test
```

Интерактивный CLI не требуется, все проверки выполняются через модульные тесты.
